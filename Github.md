# Github

github是一个免费的代码托管平台，可以帮助我们管理项目的代码

官网： github.com

## 基本概念

- 仓库（Repositroy)

  使用仓库存放项目的代码，一般项目对应一个仓库，多个项目则有多个仓库

- 收藏（Star)

  收藏仓库，方便再次查看，star的数目表示收藏的人数

- 复制克隆项目（fork）

  复制别人的项目在自己的github生成对应的仓库， 该项目是独立存在的

- pull requeest 发起请求

  向指定的仓库推送代码， 等待仓库主人审核，主人觉得代码合格则合并 

- watch关注

  关注一个开源项目，当项目发生变动时会主动通知你

- Issue事务卡片

  发现代码bug，没有成型代码，需要讨论时使用

- Github主页

  主页显示动态以及仓库

- 仓库主页

  显示项目代码，版本和收藏等信息

- 个人主页

  显示个人相关信息
